NetLabX
NetLabX is a developing software designed to model something could be represented a nodal system. The software currently only has the most basic functions for building computational graph. More functions like interpretation and routing are still unavailable and being developed.
EDA vSys
EDA vSys is a developed software for digital circuit simulation. It provides users with an environment to visually construct digital circuits. Moreover, it is convenient for users to observe the dynamic change of digital circuits in EDA vSys.
